# ApexPlanet Data Analytics Internship

## Task 1: Foundational Setup & Exploratory Data Analysis (EDA)

### Project Overview

This project was completed as part of the ApexPlanet Data Analytics Internship Program.

The objective of Task 1 was to perform data cleaning and exploratory data analysis (EDA) on the IBM Telco Customer Churn dataset to understand customer behavior, service usage patterns, and factors related to customer retention.

---

## Dataset

**Dataset:** IBM Telco Customer Churn Dataset

**Records:** 7043 Customers

**Features:** Customer demographics, subscription services, billing information, and churn details.

## Data Cleaning Performed

* Missing value analysis
* Duplicate record verification
* Data type correction
* Removal of redundant features
* Outlier detection using the IQR method
* Export of cleaned dataset

---

## Exploratory Data Analysis

The analysis includes:

* Statistical summary
* Histograms and boxplots
* Categorical comparisons using bar charts
* Correlation heatmap
* Trend analysis
* Key business insights

---

## Key Insights

1. The average customer tenure is approximately 32 months.
2. Month-to-month contracts are the most common subscription type.
3. Electronic check is the most frequently used payment method.
4. Tenure and Total Charges show a strong positive correlation.
5. Long-term customers contribute significantly higher customer lifetime value.
